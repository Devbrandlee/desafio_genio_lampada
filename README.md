# desafio_genio_lampada

<h4>O desafio proposto pelo Vai Na Web consiste em criar uma aplicação simples em React 🎉, utilizando o hook useState para gerenciar o estado de uma imagem 🖼️. Ao clicar na imagem, ela alterna entre duas versões, como uma imagem normal e uma modificada 🔄. Incluí um botão interativo para facilitar a troca de estados, proporcionando uma experiência visual ao usuário e seguindo as boas práticas do React. Essa prática foi fundamental para entender a manipulação de estados e a criação de interfaces dinâmicas.</h4>>
